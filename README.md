# 💳 Quantum Finance: Credit Score Classification & Analysis

## 🎯 Objetivo do Projeto

Como parte do time de analistas da Quantum Finance, este projeto visa explorar e modelar uma base de dados de classificação de score de crédito. O foco é transformar dados brutos em insights acionáveis e modelos preditivos que auxiliem na tomada de decisão financeira.

O projeto está dividido em duas frentes principais: Aprendizado Supervisionado (Predição) e Aprendizado Não Supervisionado/Sistemas Especialistas (Descoberta de Padrões).

## 📊 O Dataset

Utilizamos o dataset Credit Score Classification, disponível no Kaggle. Ele contém informações detalhadas sobre o histórico financeiro, comportamento de crédito e perfis de diversos indivíduos.

## 🛠️ Desenvolvimento e Metodologia

## 🔍 Parte 1: Classificação Supervisionada

O objetivo aqui é prever a categoria de score de crédito do cliente.

- EDA (Análise Exploratória): Identificação de correlações, tratamento de outliers e análise de variáveis críticas para o risco de crédito.

- Modelagem: Implementação e comparação de múltiplos algoritmos (Decision Tree, Random Forest, Logistic Regression, etc.).

- Avaliação: Uso de métricas robustas como Acurácia, F1-Score e Matriz de Confusão.

## 🧩 Parte 2: Inteligência Analítica (Não Supervisionada ou Regras)

Exploração de abordagens alternativas para gerar valor ao negócio:

- Clusterização (K-Means/PCA): Agrupamento de clientes por perfil de comportamento para identificar padrões ocultos.

- Sistema Especialista: Desenvolvimento de um motor de regras baseado em lógica de negócio para classificação automatizada.

## 🚀 Estrutura da Entrega

O projeto foi consolidado para garantir reprodutibilidade e clareza:

- [x] Jupyter Notebooks: Contendo todo o código de limpeza, EDA e modelagem.

- [x] Relatório de Performance: Comparativo entre os diferentes classificadores testados.

- [x] Documentação Técnica: Notas sobre desafios enfrentados e a utilidade prática de cada abordagem.
